# 《滿文老檔》(Manwen Laodang) 태조편 형태 주석 코퍼스

이 코퍼스는 '만문노당' 태조편 형태 주석 코퍼스입니다. 이 코퍼스의 작업에 참여한 사람은 다음과 같습니다.

### 도정업 
- 언어학 박사(서울대학교)
- 만주퉁구스어학
- (202-현재) 서울대학교 인문학연구원 선임연구원

- 정성훈. 언어학 박사(서울대학교). 코퍼스 언어학, 문법론 전공.
- 최운호. 문학 박사(서울대학교). 전산언어학, 코퍼스언어학 전공.
